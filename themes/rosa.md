# 🌸 Theme: Rosa (Cute & Soft)

Este tema está diseñado para eventos infantiles, sesiones de fotos "aesthetic" o marcas con identidad femenina y suave.

## 🎨 Design Tokens (style.css)

| Variable | Valor Sugerido | Notas |
| :--- | :--- | :--- |
| `--primary` | `#FF85A1` | Rosa vibrante para botones y títulos |
| `--accent` | `#F9C1D1` | Rosa pastel para realces |
| `--body` | `#4A4A4A` | Gris oscuro suave (menos agresivo que el azul) |
| `--hero-gradient` | `linear-gradient(180deg, #FFC2D1 0%, #FF85A1 100%)` | Degradado dulce |
| `--radius` | `24px` | Bordes muy redondeados y suaves |
| `--radius-btn` | `50px` | Botones totalmente ovalados |

## 🤖 AI Prompts (server.js)
Cambia la base de los prompts a:
- **Estilo:** "Kawaii Pixar animation style, soft clay textures."
- **Ambiente:** "Pastel pink studio, floating heart-shaped balloons, sparkling fairy dust."
- **Mascota:** "A tiny pink dragon with big sparkly eyes."

---
*Tip: Usa tipografías más redondeadas como 'Quicksand' para complementar este tema.*
