# 🐲 Spruce Spirit Photobooth

Welcome to the **Spruce Spirit Photobooth**! This is a high-fidelity AI transformation experience where you can unleash your inner dragon.

## ✨ Features

- **AI Transformation**: Uses the xAI Grok API (via Replicate) to transform your photos into legendary Pixar-style dragon characters.
- **Dynamic Gallery**: A beautiful, responsive 4-column grid showing off the latest transformations.
- **Premium Design**: Modern interface with glassmorphism, vibrant gradients, and smooth animations.
- **Fast & Responsive**: Optimized for both desktop and mobile devices.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- A Replicate API Token

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/martinezlizux/Spruce.git
   cd Spruce
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory:
   ```env
   REPLICATE_API_TOKEN=your_token_here
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

The app will be available at `http://localhost:5173` (Vite) and the backend at `http://localhost:3000`.

## 🛠️ Technology Stack

- **Frontend**: Vite, Vanilla JS, HTML5, CSS3 (Custom Design System)
- **Backend**: Node.js, Express
- **AI**: Replicate (xAI Grok Imagine model)
- **Styling**: Premium CSS with focus on aesthetics and responsiveness.
